# Arabic HTTP Studio Assets

This directory contains assets used by the application.

## Fonts

Place the following font files in `assets/fonts/`:
- Cairo-Regular.ttf
- Cairo-Bold.ttf
- Tajawal-Regular.ttf
- Tajawal-Bold.ttf

Download from:
- Cairo: https://fonts.google.com/specimen/Cairo
- Tajawal: https://fonts.google.com/specimen/Tajawal

## Icons

Place icon files in `assets/icons/`

## Images

Place image files in `assets/images/`
- logo.png (200x200)

## Note

Due to copyright, font files are not included in the repository.
Please download them manually and place them in the appropriate directories.
